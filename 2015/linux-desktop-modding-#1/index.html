Linux Desktop Modding #1

August 23, 2015

Sebelumnya, ya, saya memakai Linux. Sekarang distro yang digunakan adalah Gentoo.

Di Linux ada banyak Desktop Environment (DE), antara lain GNOME, KDE, XFCE, LXDE dan seterusnya. Nah, saya pakai DE apa? Jawabannya, tidak pakai DE.

Selain DE, ada yang namanya Window Manager (WM). Biasanya setiap DE memiliki WM sendiri. saya hanya memakai WM tanpa DE-nya. WM juga tersedia banyak pilihan, antara lain Subtle WM, Awesome WM, dan WM bawaan DE yang sudah disebutkan tadi. saya pakai WM apa? Jawabannya, Xmonad.
